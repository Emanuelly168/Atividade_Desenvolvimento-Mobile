Projeto de Notícias - React Native (Expo)

Este repositório contém a solução das atividades de desenvolvimento mobile, focando em componentes core, estilização profissional e navegação.

Funcionalidades e Telas
- Navegação por Abas: Implementada com Expo Router para alternar entre as seções.
- Home: Tela de boas-vindas com a identidade visual da marca centralizada.
- Feed de Notícias: Lista de conteúdos utilizando cards personalizados e estilizados com StyleSheet.
- Formulários de Interação: Interface para entrada de dados (texto e numéricos) seguindo o layout proposto.
